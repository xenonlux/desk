# دليل بناء تطبيق "مقر ديسك" كبرنامج مستقل (EXE) يعمل 100% أوفلاين 🖥️🔒

مرحباً بك! لقد تم تصميم وهندسة تطبيق **مقر ديسك (مصفوفة نكسيس الأمنية الموحدة)** ليعمل كـ **تطبيق سطح مكتب مستقل بالكامل دون اتصال بالإنترنت**. يتم حفظ البيانات ومزامنتها لحظياً ومحلياً على جهازك باستخدام مُحرك التخزين المحلي الآمن (`LocalStorage`) دون الحاجة لأي خوادم خارجية أو قواعد بيانات سحابية.

لإنتاج ملف تشغيلي مباشر (`.exe`) أو ملف حزمة تثبيت لويندوز (`.msi`)، نستخدم إطار العمل الفائق الأداء والسرعة والآمن **Tauri** (الذي يستند لـ Rust مع React و Vite).

---

## 📥 الخطوة 1: تنزيل ملفات المشروع على حاسوبك الشخصي
قبل أي شيء، تحتاج إلى استخراج كود التطبيق من منصة التطوير إلى جهازك المحلي:
1. من الزاوية العلوية اليمنى في منصة **AI Studio**، انقر على زر **الإعدادات (Settings)**.
2. اختر **تصدير كود المصدر (Export Project)** ثم **تحميل ملف مضغوط (ZIP)**.
3. قم بفك الضغط عن الملف في مجلد مخصص على جهاز الكمبيوتر الخاص بك (مثال: `C:\MaqarrDesk`).

---

## 🛠️ الخطوة 2: تثبيت المتطلبات الأساسية على نظام تشغيل Windows
لكي يقوم حاسوبك الخاص ببناء ولحم الكود البرمجي البرمجي وتحويله إلى برنامج ويندوز أصيل (`.exe`)، يجب توفير المتطلبات التالية (تثبت لمرة واحدة فقط):

1. **بيئة تشغيل Node.js**:
   * قم بتحميل النسخة المستقرة LTS وتثبيتها من الموقع الرسمي: [https://nodejs.org](https://nodejs.org)

2. **أدوات بناء C++ من مايكروسوفت (C++ Build Tools)**:
   * تطلبها لغة Rust لبناء الملفات الثنائية لويندوز.
   * قم بتحميل وتثبيت **Visual Studio Community** أو **Visual Studio Build Tools** من: [https://visualstudio.microsoft.com/visual-cpp-build-tools/](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
   * أثناء نافذة التثبيت، وجّه علامة الصح لتثبيت حزمة **Desktop development with C++** (تطوير تطبيقات سطح المكتب باستخدام C++).

3. **لغة البرمجة Rust**:
   * حمل أداة التثبيت الفورية لويندوز من: [https://rustup.rs](https://rustup.rs)
   * افتح الملف واضغط **1** ثم `Enter` للتثبيت بالإعداد الافتراضي الأساسي.

*(ملاحظة: يُرجى إعادة تشغيل جهاز الكمبيوتر بعد تثبيت الأدوات السابقة لتحديث متغيرات البيئة للنظام كليًا).*

---

## 🚀 الخطوة 3: البناء الفوري وإنتاج الـ EXE بنقرة واحدة!

لقد قمنا بتوفير أداة تنفيذيّة مجهّزة ومصممة خصيصاً لك لتقوم بكامل دورة البناء والكبس تلقائياً:

1. افتح المجلد الذي قمت بفك الضغط فيه.
2. ابحث عن ملف يسمى **`build_windows_exe.bat`**.
3. انقر عليه نقراً مزدوجاً (Double Click) لتشغيله.
4. سيتحقق السكريبت من برمجيات جهازك ويقوم بـ:
   * تحميل وتثبيت مجمّع مكتبات الجافاسكريبت والواجهة المستأمنة.
   * تجميع الكود البرمجي وإنتاج نسخته المحسنة المغلقة (`dist`).
   * تشغيل لغة Rust ومحرك Tauri لتشييع وبناء الكود البرمجي في ملف ويندوز موحد وبسرعة فائقة.

---

## ⌨️ الطريقة اليدوية للبناء عبر موجه الأوامر (Command Prompt):
إذا كنت تفضل استخدام الـ Terminal بنفسك، افتح موجه الأوامر داخل مجلد المشروع ونفذ الأوامر التالية بالتتابع:

```bash
# 1. تثبيت الاعتماديات والمكونات البرمجية المعتمدة
npm install

# 2. بناء واجهة الويب المحسنة بالكامل أوفلاين
npm run build

# 3. تجميع وبناء تطبيق سطح المكتب EXE المستقل لنظام ويندوز
npm run tauri build
```

## 📁 أين سأجد ملف التطبيق (.exe) المكتمل؟

عند انتهاء معالج البناء بنجاح، ستجد الملفات في المجلدات التالية داخل مجلد مشروعك الرئيسي:

* **ملف البرنامج المباشر (Standalone Portabile EXE)**:
  `C:\MaqarrDesk\src-tauri\target\release\مقر ديسك - حجز وإدارة المساحات.exe`
  *(يمكنك نسخ هذا الملف ونقله لأي فلاش ميموري وتشغيله فوراً على أي جهاز يعمل بنظام ويندوز دون اتصال بالإنترنت)*.

* **ملف حزمة التثبيت الأوتوماتيكية (Setup Installer / MSI)**:
  `C:\MaqarrDesk\src-tauri\target\release\bundle\msi\`
  *(وهو ملف تثبيت تنفيذي ممتاز جاهز يتم تثبيته في قائمة إبدأ لجهازك ويوفر أيقونة تفاعلية على سطح المكتب).*

## 🔒 آلية التفعيل الأمني بدون إنترنت (Offline Safe Activation)
يعمل التطبيق بمصفوفة حماية كاملة ومحلية مستقلة عن الإنترنت:
* عند تشغيله لأول مرة، يطلب كود الفك الماستر المستقر للعمل بصفة دائمة.
* كود الفك السريع الفوري والمفتاح المعتمد الموثق للمحطة هو:
  🔑 **`تفعيل-نكسيس-الآمن`**
* يمكنك تدوينه بالكامل والدخول فوراً، وسيتم حفظ التفعيل على جهازك للأبد وبدون إنترنت كلياً!
